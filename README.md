# EchoExtends (React + Python)

Reactで作成した6x6のターン制ボードゲームです。

## 現在の進捗
Reactで、簡単なPvPを同デバイス上で行うことができる。

## 今後の予定
-UIを整える  
-PythonでAIの学習を開始する  
-対AI機能を実装する

---

## ルール
-プレーヤーは以下のルールに沿って交互にマスを塗る  
-最初のターンでは好きなマスを１つ塗ることができる  
-次のターン以降は、相手のマスを1つ選び、縦横の4方向から1方向を選び　　
空きマスがあれば、1マス、または、2マス伸ばすことができる  
-これを繰り返し、盤面がすべて埋まったときに多く塗れていたプレイヤーの勝ち

## 備考
-相手のマスを塗り替えることはできない  
-すでに塗られたマスを飛び越えてマスを塗ることはできない  
-マスを塗ることが不可能な場合に限り、パスとなり相手のターンとなる
